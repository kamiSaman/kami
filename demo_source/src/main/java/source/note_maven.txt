1、eclipse安装maven
	1) 安装meeclipse插件，Help->Eclipse Marketplace搜索Maven Integration for Eclipse点击安装  (有插件可省略)
	2) 安装maven
		1> maven官网：http://maven.apache.org/download.cgi
		2> 配置环境变量
			MAVEN_HOME=C:\utils\apache-maven-3.6.1
			PATH=%MAVEN_HOME%\bin
	3) eclipse集成maven
		1> windows->preferences-->maven->installations,配置maven路径
		2> 修改maven配置文件windows->preferences-->maven->User Settings
			Golbal Settings：所用工作空间的配置文件
			User Settings  ：当前空间配置文件
	4> 设置maven配置文件，修改settings.xml
		1> 修改本地仓库地址:<localRepository>C:\utils\apache-maven-3.6.1\mvn_repository</localRepository>
		2> 访问仓库地址：
			<mirror>
				<id>alimaven</id>
				<name>aliyun maven</name>
				<url>http://maven.aliyun.com/nexus/content/groups/public/</url>
				<mirrorOf>central</mirrorOf>
			</mirror>
